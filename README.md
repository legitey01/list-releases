# List Releases

This repository hosts the official releases for **List** — an AI-powered code editor built on VS Code.

## Download

Go to the [Releases](../../releases) tab to download the latest version for your platform.

| Platform | File |
|---|---|
| Windows (x64) | `List-Setup-x64.exe` |
| Windows (arm64) | `List-Setup-arm64.exe` |
| macOS | `List.dmg` |
| Linux | `List.AppImage` |

## Auto-Update

List checks for updates automatically on startup and every 3 hours.  
When a new version is available, it downloads in the background and notifies you before restarting.
